一、新建&提交（这里我用的是gitBash.gif）
1. git config --global user.name "你自己的github的名字"
2. git config --global user.email "你自己的github的邮箱"

以下内容全部在所需上传的大文件夹中操作
1. git init                                  （初始化git）
2. git add .                                 （添加文件到git）
3. git commit -m "提交文件"                  （提交文件）
4. git remote add origin 要提交的仓库地址     
5. git push -u origin 分支名字                （分支名字可为默认的master也可为自定义名字，但如果分支已存在则无法新建并提交）
